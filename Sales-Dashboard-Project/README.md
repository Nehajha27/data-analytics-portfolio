# Sales Performance Dashboard

## Overview
This project analyzes sales data to identify trends, top-performing products, regions, and customers.
It demonstrates end-to-end data analysis using SQL, Excel, and Tableau, with a focus on business insights.

## Business Questions Answered
- How are sales and revenue trending over time?
- Which products and regions generate the most profit?
- Which customers contribute most to overall sales?
- Where are opportunities to improve performance?

## Tools Used
- SQL for data analysis and aggregation
- Excel for data cleaning and preparation
- Tableau for interactive dashboards and visualization
